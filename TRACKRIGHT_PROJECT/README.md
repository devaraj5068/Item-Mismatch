# TrackRight Project

A modern, animated, industry-ready web application for inventory management and order tracking.

## 🚀 Animated Features

### Global Animations
- **Page Load Fade-in**: Smooth entry for all pages
- **Section Scroll Reveal**: AOS-powered reveals on scroll
- **Button Interactions**: Ripple effects, hover glow, pulse animations
- **Card Lifts**: Hover transformations with shadows
- **Smooth Transitions**: Between pages and states

### Dashboard
- **KPI Cards**: Count-up animations for metrics
- **Interactive Charts**: Chart.js with load animations
- **Real-time Feel**: Skeleton loaders and shimmer effects
- **Toast Notifications**: Success/error feedback with auto-dismiss

### Order Flow Visualization
- **Animated Timeline**: Step-by-step progress (Ordered → Picked → Packed → Shipped → Delivered)
- **Pulsing Stages**: Current stage highlights
- **Error Flashes**: Red animations on mismatches

### Scan & Verify
- **Scanner Frame**: Animated scanning line
- **Success Feedback**: Tick animations and vibrations
- **Confidence Meter**: Radial progress bar for match confidence
- **AI-Ready**: Placeholder for future ML integrations

### Mismatch Reports
- **Animated Filters**: Slide-in/out filter panels
- **Row Highlights**: Critical errors with shake animations
- **Drill-down Charts**: Interactive data exploration

### Smart Alerts & Feedback
- **Modal Alerts**: Animated popups for notifications
- **Severity Colors**: Green/Yellow/Red based on issue level
- **Micro-interactions**: Hover tooltips, focus glows

## 🏗️ System Architecture

- **Frontend**: Vanilla JS with AOS, Chart.js, CSS Animations
- **Backend**: Django REST API with Token Authentication
- **Database**: SQLite (easily switchable to MySQL)
- **Deployment**: Vercel (frontend), Heroku/AWS (backend)

## 📊 Analytics & Insights

- Scan verification with confidence scoring

## 🛠️ Setup

### Backend (Django)
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python manage.py migrate`
4. `python manage.py createsuperuser`
5. `python manage.py runserver`

### Frontend
1. `cd frontend`
2. `python -m http.server 3000`
3. Open http://localhost:3000
